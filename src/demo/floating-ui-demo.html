NovelSynth Floating UI Demo

About the Floating UI

The NovelSynth Floating UI provides quick access to AI enhancement features directly on web pages. It appears as a small button in the bottom right corner that expands into a panel with controls.

  • Draggable: The button can be moved anywhere on the screen
  • Context-aware: Automatically detects the type of content (novels, news, learning)
  • Quick access: One-click enhancement, summarization, and analysis
  • Custom settings: Adjust temperature, model, and other parameters
  • Keyboard shortcuts: Alt+E (enhance), Alt+S (summarize), Alt+A (analyze)

Floating UI Components

Main Button

The floating button appears in the bottom right corner of the page. It can be dragged to any position on the screen.

Main Panel

Clicking the button opens a panel with quick actions and settings. The panel includes:

  • Content type badge (shows detected content type)
  • Action buttons (Enhance, Summarize, Analyze)
  • Quick settings (Temperature, Model)
  • Progress indicator (when processing)

Settings Panel

Clicking the settings icon in the panel header shows the settings view. This includes:

  • Prompt settings (Content type, Custom prompt)
  • Advanced options (Chunking, Emoji, Max tokens)
  • Link to full settings

Integration with Main Extension

The floating UI complements the main extension popup by providing quick access to the most common features. Settings are synchronized between both interfaces.

Key differences:

  • Popup UI: Comprehensive settings, library management, and advanced options
  • Floating UI: Quick access to core features with context-specific settings

The floating UI detects the website type and automatically selects the appropriate prompt template, making it easier to get relevant results without manual configuration.

Usage Instructions

  1. Navigate to any website with text content
  2. The floating button will appear in the bottom right corner
  3. Click the button to open the panel
  4. Select an action (Enhance, Summarize, or Analyze)
  5. The result will appear in a floating window

You can also use keyboard shortcuts:

  • Alt+E: Enhance content
  • Alt+S: Summarize content
  • Alt+A: Analyze content
