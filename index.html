Installation

1

Download the Extension

Download the latest release from the releases page or build from source.

Download Release Build from Source
2

Install in Browser

Chrome/Edge

  1. Open Chrome Extensions (chrome://extensions/)
  2. Enable "Developer mode"
  3. Click "Load unpacked" and select the extension folder

Firefox

  1. Open Firefox Add-ons (about:addons)
  2. Click the gear icon and select "Debug Add-ons"
  3. Click "Load Temporary Add-on" and select manifest.json
3

Configure AI Providers

Add your API keys for supported AI providers in the extension popup.

Get Gemini API Key Get OpenAI API Key Get Anthropic API Key
